ESD Metrics Comparison Dashboard

This dashboard provides direct comparisons of Empirical Spectral Density (ESD) metrics across different dataset sizes. These comparisons can help understand how dataset size affects the learning dynamics and generalization properties of neural networks.

Dataset Sizes Compared: 2000, 5000

Key Metrics:

  • Alpha: Power-law exponent of the eigenvalue distribution. Lower values indicate more complex representations.
  • D: Kolmogorov-Smirnov statistic measuring the goodness of fit of the power-law distribution.
  • Spectral Norm: The largest eigenvalue of the weight matrix, indicating the maximum amplification the layer can apply.
