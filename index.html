Deep Grokking ESD Analysis

This project explores the relationship between Empirical Spectral Density (ESD) metrics and neural network learning dynamics.

ESD metrics provide insights into the eigenvalue distributions of weight matrices in neural networks, which can help understand how networks learn and generalize.

Run New Analysis

To run a new analysis with different parameters, use the following command:

python analyze_esd.py --dataset_size SIZE --metrics_dir metrics --output_dir OUTPUT_DIR

To train a model with a different dataset size:

python -c "from grokking import train_and_visualize; train_and_visualize(train_dataset_sizes=[SIZE], max_steps=10000, step_size=1000)"

Key Metrics Explained

  • Alpha: Power-law exponent of the eigenvalue distribution. Lower values indicate more complex representations.
  • D: Kolmogorov-Smirnov statistic measuring the goodness of fit of the power-law distribution.
  • Spectral Norm: The largest eigenvalue of the weight matrix, indicating the maximum amplification the layer can apply.
